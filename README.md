
# The Flap 🦋

> "A single event can alter the course of a system forever."

An interactive visual exploration.


### Live Demo
[Click here for Live Demo](https://kristina-neureuther.github.io/the-flap/)

### Tech Stack
- HTML5 Canvas & Vanilla CSS
- p5.js (Creative Coding Library)

### How to Run
1. Open the live deployment or run `index.html` locally.
2. Press the **SPACEBAR** or **CLICK** anywhere on the canvas to trigger a flap and witness the immediate structural transformation of the system.
